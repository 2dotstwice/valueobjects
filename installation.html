ValueObjects - Installation

Using Composer, require the nicolopignatelli/valueobjects package:

{
    "require": {
        "nicolopignatelli/valueobjects": "~1.1"
    }
}

Otherwise, install the library and setup the autoloader yourself.

Running the tests

Install the dev dependencies from Composer:

php composer.phar install --dev

Then, run the test suite:

bin/phpunit
