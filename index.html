ValueObjects

A PHP 5.3+ library/collection of classes aimed to help developers with domain driven development and the use of immutable objects.

Covered types

ValueObjects should cover every common type a developer could need in a DDD project. With this goal in mind, the object library should stay as generalistic as possible in order to be easily extendable and customizable in every single project.

Types list (mostly implemented as namespaces):

  • [x] Dates and time
  • [x] Null, Numbers and Strings
  • [x] Data structures
  • [x] UUID
  • [x] Person data (age, gender, etc...)
  • [x] Money and currencies
  • [x] Geographic data (coordinates, addresses, etc...)
  • [x] Internet related (URLs, email addresses, IPs, etc...)
  • [ ] Submit yours!
